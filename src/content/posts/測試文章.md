這是一篇測試文章~!
Hello World
```
name: Sync Blog Folder to Fuwari

on:

push:

paths:

- '6.Blog/**' # Triggers only when files in your blog folder change

  

jobs:

build-and-deploy:

runs-on: ubuntu-latest

steps:

- name: Checkout Private Vault

uses: actions/checkout@v4

with:

fetch-depth: 0 # 獲取完整歷史紀錄以確保同步正確

  

- name: Pushing to Fuwari Repo

uses: cpina/github-action-push-to-another-repository@main

env:

API_TOKEN_GITHUB: ${{ secrets.BLOG_SYNC_TOKEN }}

with:

source-directory: '6.Blog'

destination-github-username: 'livingfree2023'

destination-repository-name: 'fuwari'

user-email: github-actions[bot]@users.noreply.github.com

target-directory: 'src/content/posts'

target-branch: main
```
